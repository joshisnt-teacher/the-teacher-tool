# Year 9 Digital Technology — SCSA Alignment Note

Year 9 Digital Technology at Rehoboth Christian College is a **school-designed elective course**.

It does not align to the SCSA Digital Technologies curriculum and has no official SCSA descriptor codes assigned.

When creating or tagging resources for this course, leave `scsa_descriptors: []` in the resource card.

---

## Course Units

The elective is divided into four unit categories:

| Unit | Strand shorthand | Description |
|---|---|---|
| 3D Printing | `3DPrint` | CAD design (OnShape) and physical fabrication (Bambu Lab) |
| Game Design | `GameDesign` | Game development using Godot engine |
| Robotics | `Robotics` | Physical computing and robotics projects |
| Python Programming | `Python` | Text-based programming using Python |

---

## Naming Convention

Use the unit category as the strand field in filenames:

```
Y9-DigiTech-3DPrint-[Topic]-[Subtopic].pptx
Y9-DigiTech-GameDesign-[Topic]-[Subtopic].pptx
Y9-DigiTech-Robotics-[Topic]-[Subtopic].pptx
Y9-DigiTech-Python-[Topic]-[Subtopic].pptx
```
